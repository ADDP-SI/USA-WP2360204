```python
# Membuat tiga variabel
nama_produk = "Laptop Gaming ASUS ROG"
harga = 15000000
stok_tersedia = 5

# Mencetak ketiga variabel ke layar
print("Nama Produk:", nama_produk)
print("Harga: Rp", harga)
print("Stok Tersedia:", stok_tersedia)

```

Alternatif dengan format yang lebih rapi:

```python
# Membuat tiga variabel
nama_produk = "Laptop Gaming ASUS ROG"
harga = 15000000
stok_tersedia = 5

# Mencetak dengan format yang lebih baik
print(f"Nama Produk: {nama_produk}")
print(f"Harga: Rp {harga:,}")
print(f"Stok Tersedia: {stok_tersedia} unit")
```
